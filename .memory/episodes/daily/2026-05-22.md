### Decisions Made

- **편/장 모두 H1**: 실제 한국 법령에서 편과 장은 같은 문서에 최상위로 공존하지 않으므로 같은 버킷에 배정해도 황금 파일 회귀 없음. 기존 moel_01-05 변화 없음.
- **관(subsection) 계속 연기**: 일반 명사 '관'과 충돌, 실제 픽스처 없음.
- **heading_style.rs 추출 방식**: `convert.rs`에서 `pub(crate) use`로 재수출, `convert_tests_detect.rs`에서 `is_heading_terminator`는 직접 import. 이유: CI `-D warnings`에서 미사용 import 오류 방지.
